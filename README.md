# How to run 

```
# 가상환경 세팅
	conda create --name=django-with-react-rev2 python=3.8
# 가상환경 접속
  conda activate django-with-react-rev2

# 설치 확인
  conda env list 
  # mac
    which python | which pip
  # window
    where python | where pip

# Django 설치
pip install django~=3.0.0

# Project 생성
django-admin startproject askcompany .
```
